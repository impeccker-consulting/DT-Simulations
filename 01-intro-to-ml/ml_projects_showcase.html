🔬 Technology Glossary — Plain English

Cost tiers: 🆓 Free / Open Source 💛 Freemium 💰 Paid / Commercial
Machine Learning (ML)
🆓 Free
Teaching computers to learn from data and make predictions — like how Netflix knows what you'll watch next. Libraries like scikit-learn & PyTorch are free.
LangChain
🆓 Free
A toolkit to build AI-powered apps that can hold conversations, search the web, and take actions. The library itself is free; LLM API calls cost money.
RAG (Retrieval Augmented Generation)
🆓 Free
Give an AI your company's documents so it answers from YOUR data. It's a design pattern — free to implement using open-source tools.
NLP (Natural Language Processing)
🆓 Free
Teaching computers to understand human language — emails, reviews, chats. spaCy, NLTK, Hugging Face are free. Premium APIs cost money.
Vector Database
💛 Freemium
Smart search by meaning. FAISS & ChromaDB are completely free. Pinecone has a free tier; paid plans start ~$70/month for production scale.
LLM (Large Language Model)
💛 Freemium
AI models like GPT-4 or Claude. OpenAI API: ~₹0.8–2.5 per 1K words. Open-source models (Llama, Mistral): completely free to run yourself.
MLOps
💛 Freemium
Tools to deploy & manage ML models. MLflow & DVC are free open-source. Cloud MLOps (AWS SageMaker, Azure ML): pay-per-use from ~$0.10/hr.
Streamlit / Gradio
🆓 Free
Build ML dashboards and demos in Python — no web developer needed. Both are free locally & have free cloud hosting tiers for student projects.
Computer Vision
🆓 Free
Teaching AI to "see" using free libraries (YOLOv8, OpenCV, PyTorch). Main cost is GPU cloud compute — can use free Google Colab for projects.
Pinecone / ChromaDB / FAISS
💛 Freemium
Vector databases for fast AI search. FAISS & ChromaDB: completely free. Pinecone: free 1 index, paid from ~$70/month for large datasets.
Apache Spark / Kafka
💛 Freemium
Process millions of data records in real time. Both are free open-source. Managed cloud versions (Confluent Cloud, Databricks): paid per usage.
Power BI / Tableau
💰 Paid
Business dashboards for management. Power BI: free desktop app, Pro ₹750/user/month. Tableau: starts ~$15/user/month. Both have free student versions.
